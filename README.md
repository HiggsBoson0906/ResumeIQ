# ResumeIQ

AI-powered resume analysis platform that scores, gaps, and optimizes your resume before an ATS ever gets the chance to reject it.

## Tech Stack

- **Frontend**: React, Vite, React Router
- **Backend**: Node.js, Express, MongoDB, Mongoose
- **AI**: Google Gemini API

## Getting Started

### Prerequisites

- Node.js (v18+)
- MongoDB

### Installation

```bash
# Install all dependencies
npm install

# Start backend
cd backend
npm run dev

# Start frontend (new terminal)
cd frontend
npm run dev
```
